## Current Features

* Homepage
* Reader's Guide
* Chapter companions
* Character pages
* Themes

---

## Project Goals

Read Cameron Reed is designed to help readers engage more deeply with Cameron Reed's work through thoughtful companion content.

The site focuses on:

* Chapter-by-chapter reading guides
* Character references
* Themes and literary analysis
* Worldbuilding notes
* Reading resources

The goal is to complement the reading experience—not replace it.
